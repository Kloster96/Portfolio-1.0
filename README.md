# 🌐 Mi Portfolio Personal

Mi portfolio personal construido con **Next.js 15**, **Tailwind CSS** y herramientas web modernas.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: [Tailwind CSS 3](https://tailwindcss.com/)
- **TypeScript**: Componentes y utilidades completamente tipadas
- **Animaciones**: GSAP, Lottie
- **Icons**: [Lucide](https://lucide.dev/), [React Icons](https://react-icons.github.io/)

---

## 🚀 Getting Started

### 1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/tu-portfolio.git
cd tu-portfolio
```

### 2. Instalar dependencias

```bash
pnpm install
# o
npm install
```

### 3. Ejecutar el servidor de desarrollo

```bash
pnpm dev
```

Visitá: [http://localhost:3000](http://localhost:3000)

---

## 📋 Configuración

Edita los datos personales en:

- `utils/Data/PersonalData.ts` - Tu información de contacto
- `utils/Data/projects-data.ts` - Tus proyectos
- `utils/Data/experience.ts` - Tu experiencia laboral
- `utils/Data/educations.ts` - Tus estudios
